GEOUNED
=======

.. toctree::
   :maxdepth: 2
   :caption: Contents:

GEOUNED converts CAD to Constructive Solid Geometry (CSG) formats for use in
Monte Carlo transport codes.
Supported codes include OpenMC, PHITS, Serpent and MCNP.
